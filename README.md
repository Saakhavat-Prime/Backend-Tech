# Backend Tech

A simple backend learning repository built with **Node.js** and **Express.js**. This repository contains backend practice projects and concepts that will be updated as I continue learning.

## 🚀 Tech Stack

- Node.js
- Express.js
- JavaScript
- Nodemon

## 📦 Installation

```bash
git clone https://github.com/your-username/Backend-Tech.git
cd Backend-Tech
npm install
npm run dev
```

## 📚 Topics

- Node.js
- Express.js
- Routing
- Middleware
- REST APIs
- CRUD Operations
- Authentication
- MongoDB
